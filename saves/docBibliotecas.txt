USER: como usar a pdf-parse 
ASSISTANT: A pdf-parse é uma biblioteca JavaScript que permite extrair texto de arquivos PDF. Aqui está um exemplo básico de como usá-la:

Primeiramente, você precisa instalar a pdf-parse usando o npm:
```
npm install pdf-parse
```
Depois, você pode usar a seguinte código para extrair o texto de um arquivo PDF:
```javascript
const pdf = require('pdf-parse');

const pdfFile = 'caminho_do_arquivo.pdf';

pdf(pdfFile).then(data => {
  const texto = data.text;
  console.log(texto);
}).catch(err => {
  console.error(err);
});
```
Nesse exemplo, você passa o caminho do arquivo PDF como argumento para a função `pdf()`. A função retorna uma promessa que resolva com um objeto que contém o texto extraído do arquivo PDF.

Você também pode usar a pdf-parse para extrair texto de um arquivo PDF que está armazenado em uma variável, em vez de ler ele de um arquivo:
```javascript
const pdf = require('pdf-parse');
const fs = require('fs');

const pdfBuffer = fs.readFileSync('caminho_do_arquivo.pdf');

pdf(pdfBuffer).then(data => {
  const texto = data.text;
  console.log(texto);
}).catch(err => {
  console.error(err);
});
```
Além disso, você também pode configurar opções para a extração de texto, como o idioma e a codificação:
```javascript
const pdf = require('pdf-parse');

const pdfFile = 'caminho_do_arquivo.pdf';
const options = {
  // Configura o idioma para o texto extraído
  lang: 'pt',
  // Configura a codificação para o texto extraído
  encoding: 'utf8'
};

pdf(pdfFile, options).then(data => {
  const texto = data.text;
  console.log(texto);
}).catch(err => {
  console.error(err);
});
```
Essas são apenas algumas das opções disponíveis para a pdf-parse. Você pode verificar a documentação oficial para obter mais informações sobre como usar essa biblioteca.
